copy qiqqa library to another computer

Dear,

I have exported my qiqqa library and copied it to a faster computer, where I also installed qiqqa.

The PDFs were automatically exported to a folder 'docs', the tags were exported to a folder 'tags', the BibTex references were put in a BibTex file, etc.

I was able to import my PDF files, but now I would like to restore the tags and BibTex references to the corresponding documents.

However, I don't know where I can copy these folders to, so that qiqqa can find them.

    The easiest and fastest way to transfer data between computers is to use the backup and restore functionality.

    In the old computer open Qiqqa and log out. (Lower left hand corner of the screen). Then open Qiqqa. Instead of logging in, click on "Backup/Restore" and select Backup. This will create a zip file which has everything in in.

    Move the backup file to your new computer (eg thumb drive).

    On the new computer open Qiqqa. If you've already logged in before, you will have to logout and reopen as above to get to the Backup/Restore button. Use the Restore option to access the backup file. Qiqqa will create an exact copy of the data as it was on the old computer.

    WARNING: Restore will destroy any Qiqqa data on the new computer. This is generally what you want if your are copying to a new computer, but be careful.
