Search function not working

Hi,

I have been using Qiqqa (v.76) nearly every day for my PhD work. Today suddenly I find that the search function does not work at 'home' or 'library' level. Search only works within pdf.

I went to Qiqqa help and tried reindexing after selecting all pdfs, as suggested in one of the posts. Nothing happened. Tried logging out and restarting. No improvement. Tested out even simple words like 'research' - 0 results except within a pdf. Even tried selecting all pdfs and launching search at library level - 0 results again.

Folks, this is seriously not a good time for this to be happening to me. In the middle of working papers for publication and organising final draft of PhD.

HELP!!!

Shailaja

    maadri mosharef over 3 years ago
    Despite the thread is quite old and if anyone is still facing the same issue. If the rebuild indices under qiqqa configuration does not work. try to go to your library, select all documents (CTRL+A), then right-click > misc > reindex this document.
    Please refer to Qiqqa Manual, page 15.
