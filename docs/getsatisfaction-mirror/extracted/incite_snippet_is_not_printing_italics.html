Incite snippet is not printing italics

When I copy a snippet and then print to Word there are no italics.
This is the case for every style.
It is only happening on one computer. (It operates correctly on another computer that I have qiqqa synced to)
Is there some setting I have inadvertently set wrong?

    ArthurHB over 8 years ago
    I have determined that is definitely not a Qiqqa problem. It has to do with settings in Word and also other Word add-ons taking control of formatting when pasting.
    All solved.
