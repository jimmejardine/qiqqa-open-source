How do I load an exported library`?

I exported a qiqqa library from another computer, want to load it on qiqqa on this computer. How do I do that? There is no file selector in the library view.
Please don't tell me I have to go via online storage to load a library which is already on the local computer.

PS; How about a searchable manual? Video instructions are neat, but useless for quickly finding help on specific topics.

    Mike Noren over 8 years ago
    OK, I found the "add from import" function, and it's broken.

    It imported the metadata but not the PDF's, forcing me to manually import and then "link" each PDF in the library with the metadata. Did I mention I have 600 PDF's in my library?

    I checked, and the first problem is that Qiqqa uses absolute paths in the qiqqa.bib file (that's a very bad idea, use the volume name or ask the user!). Since my USB stick is drive D: on one computer and drive E: on the other, Qiqqa fails to find the PDFs. Using a texteditor I changed all D:'s to E:'s and tried again. No joy, Qiqqa still can't find the PDFs.

    I can only conclude that importing a qiqqa library in to qiqqa presently does not work.

    Qiqqa (Official Rep) over 8 years ago
    Hi there - we really don't recommend copying PDFs between two Qiqqa instances using the export/import feature. Import is purely intended for importing from other software products. Export is purely intended for those who was peace of mind that they can migrate away from Qiqqa.

    By exporting and importing, you lose all the additional benefits that Qiqqa offers because it has to squeeze everything through the generic PDF format - so you lose nice editable tags, annotations, highlights. They are viewable in the PDF files, but no longer editor in the powerful Qiqqa way.

    Rather - to copy from one computer to another, either sync via the cloud, or copy the actual underlying Qiqqa data directory. You can get there from the config screen...

    Ryan Armstrong over 8 years ago
    This seems to be a recurring issue, and it is rather disappointing that Qiqqa does not provide an efficient facility to move PDFs and metadata between computers through data export/import.

    Qiqqa (Official Rep) over 8 years ago
    If you want to manually move the Qiqqa database, all you need do is zip up the Qiqqa data directory and unzip it on the target computer.

    The efficient means of keeping two computers in sync is using the sync built into Qiqqa :-)
