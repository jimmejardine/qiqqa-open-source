Importing EndNote library to Qiqqa

I recently bought a new computer, but was unable to locate my EndNote software disk to reinstall the program. I did however, save all my EndNote library files.

I am thinking of switching from EndNote X5 to Qiqqa (and if it does what it says it does, its a better choice anyway) - how would I import my EndNote library (only about 300 references right now), and should I anticipate any problems?

Right now the majority of references do not have attached pdfs, but some do. Will the import process also include the attached pdfs?

    Hi Nikki.

    Qiqqa can current import from Endnote. But you need to EXPORT from Endnote first (Qiqqa doesn't read the library files directly).

    So you'll need to ask a friend to open up your libraries and export them (all the instructions on how to do this are in the Qiqqa app when doing the import). You can then import those export files.

    It does import the PDFs yes.
