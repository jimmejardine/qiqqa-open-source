superscript, subscript, italics, and bold formatting with Bibtex and reference list

How to format superscript, subscript, italics, and bold formatting in your Bibtex reference:

  • Superscript example:    <sup>3</sup>H    <sup>14</sup>C
  • Subscript example:       H<sub>2</sub>O 
  • Italics example:             <i>Cupaniopsis anacardiodes</i>
  • Bold example:              <b>this is bold</b>

HTML tags (i.e. the <sup> and </sup> before and after the text to be formatted) are used to format the references

Make sure there are no spaces between the text being formatted, e.g. 2, and the tags, e.g. <sub> and </sub>, in H<sub>2</sub>O

For more HTML formatting tags: http://www.simplehtmlguide.com/text.php
