tag management

Qiqqa is a great program - from the perspective of qualitative data analysis (QDA) programs it's one that specialises in literature review tasks. But it could be extended to other kinds of text, and can draw on QDA programs for some ideas. Tags are like "codes" in (QDA). It's important to manage codes to ensure consistency in use. One way is to be able to associate some canonical text / example with a tag/code name. This would be a good addition to Qiqqa.
