"Contrating" (versus expanding) in Brainstorm

When exploring themes in a brainstorm, I'd like to be able to expand the most influential documents, and then contract them and select the most specific documents. and vice versus. Also, it would be nice if there were some way of distinguishing between the two types when both types are displayed.

    To "almost simulate" this request, did you know you can select a whole batch of nodes by holding down the SHIFT key while dragging a box over the nodes.

    Then hit DEL to delete them.

    TCS over 6 years ago
    I'm surprised that something like this has not been implemented yet.

    If i select a resources and then hit T to expand all tags, why not reverse the process with I hit T again? This would be a very nice addition. Hit T, G, S, or whatever - and if it's not expanded, expand it. If it is expanded, then contract it (delete). Functionality like this would make navigation much more fluid and intuitive - and especially needed for any large libraries.
