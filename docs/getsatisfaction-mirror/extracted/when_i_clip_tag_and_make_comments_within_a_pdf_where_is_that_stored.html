Andrew Heyd 1 year ago

When I clip, tag, and make comments within a PDF, where is that stored?

How do I ensure I can back that up so it doesn't get lost?
7796998
  • 0
    Dennis 1 year ago
    Any metatdata (highlights, tags, annotations, comments, etc) made in the Qiqqa pdf view are stored in that library's Qiqqa.library file which is an sqlite3 database.

    See https://getsatisfaction.com/qiqqa/top...

    Qiqqa has built in cloud sync for backup, but I would recommend doing your own backup of your Qiqqa data files for added assurance.

    The easiest way to backup is to backup everything into a zip file with the "Backup/Restore" command under Tools in the Home tab.

    Do not try to do automatic backup of live Qiqqa files with Dropbox or similar solutions, as that has been known to corrupt the library. It is safe to put the zip files from a backup in Dropbox etc.

    Dennis 1 year ago
    By "live qiqqa files", I meant the files that qiqqa is actively updating. Examples of these would be the sqlite library database and the files used by the Lucerne text search engine. The pdf's, once added, are treated as read only and should be relatively safe for Dropbox storage.
