    Hi Patrick,

    Just to clarify... A snippet is an option in Qiqqa (see attacked pic). When you select "snippet" a copy of the reference is stored in memory - it behaves just as if you copied text in word.

    Once you select snippet, if you can move into the word document and select "paste" (or CTRL-V), which results in having the complete reference pasted into the document.

    If you paste into a program that does not have text formatting (programs that do not support RTF, eg NotePad, etc) then the reference will not contain any formatting (eg Italics)
