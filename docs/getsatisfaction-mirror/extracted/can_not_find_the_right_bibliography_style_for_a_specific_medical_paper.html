Can not find the right bibliography style for a specific medical paper.

I want to publish a review study in a paper called KSSTA. The problem is that I couldn't find the bibliography style this paper uses and in the instruction for publishing the website offers options for zotero, endnote and sente. What should I do? I really like qiqqa and wouldn't like to switch to a different reference manager!!!
6416882
  • 1
    Dennis over 6 years ago
    I went to look at the KSSTA website. When Google found it, it warned that the site may have been hacked and recommended avoiding it, so I did.

    A generic reply is that because both Qiqqa and Zotero use the same styles (CSL), you can use the ones from Zotero.

    With the Qiqqa "InCite" tab open, select the "Citation Style" dropdown and choose "Browse CSL Library". This opens the Zotero Style Repository. A search on "KSS" finds the style "Knee Surgery, Sports Traumatology, Arthroscopy" which sounds like what you are looking for.

    By the way, that style is already in the Qiqqa styles of my version 67s, though it is dated 8/29/2014 as opposed to the Zotero version which is dated 9/6/2014.

    Hope this helps.
