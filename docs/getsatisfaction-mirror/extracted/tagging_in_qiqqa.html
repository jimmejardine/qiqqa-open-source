Tagging in Qiqqa

Hello
Help me with this..

When I add files from my computer into the library, It is not sorted in the form of folders as I previously sorted them in windows. Am I supposed to go through the library again and tag each paper and then those papers will appear under the relevant tag?? Is it so?

  • EMPLOYEE

    Qiqqa (Official Rep) over 7 years ago
    If you drag a single PDF into Qiqqa then this is the case. It will not add any tags.
    If you use the "add folder" option from the library screen, then you can select "add subfolder names as tags".
