    Tanya Murphy over 4 years ago
    I got a ton of stupid tags (digits, of, the, a) I think from importing a couple of thousand entries from Mendeley (its automated inclusion of keywords is terrible).

    A bulk edit procedure that sped things up for me a little was to:

    1. select a bunch from the left tag list (manually clicking each box)
    2. right-click in the top area starting with "Library currently filtered by:..."
    3. choose select all and copy
    4. paste into Notepad and do a Replace... with Find = " OR " and Replace = ;
    5. select all of the documents in the centre panel
    6. paste the semi-colon separated list from Notepad into the empty tag slot in the Tags - Remove Multiple area in the left panel ENTER
    7. press apply
    8. refresh tag list

    Some relief...but it shouldn't be this difficult!
