    Libraries can be deleted or renamed from the web interface by logging in at Qiqqa.com, opening the library of interest, selecting "Library settings" from the menu.

    Changes made on line will be reflected on desktop(s) after syncing.

    Deleted libraries will be greyed out in Qiqqa desktop and can then be "purged" to make them go away.
