How do I create a vanilla reference in PubMed?

4322722
  • 0
    Dennis over 8 years ago
    1) Create a empty vanilla reference in Qiqqa (Menu: Add non-PDF reference under Add PDF's or References)
    2) With the empty vanilla reference open, select Sniffer just below the blank Bibtex data box on the right.
    3) When Sniffer window opens choose the Pubmed tab.
    4) Search for your article and select it when found (should open in abstract view)
    5) Use the "Display settings" dropdown to change the view from Abstract to XML
    6) The Bibtex data is now imported automatically from the xml view
    7) Done - can close the Sniffer window.
